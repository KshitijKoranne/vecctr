# Vecctr

**Beautiful backgrounds, instantly.**

A generative SVG design asset tool built under [KJR Labs](https://kjrlabs.in).

## Generators

| Generator | Tier | Description |
|-----------|------|-------------|
| Surge | Free | Organic wave layers with noise distortion |
| Aura | Free | Dreamy mesh gradients with soft color orbs |
| Morph | Free | Organic blob shapes, positioned and scaled |
| Drift | Free | Scattered soft shapes floating across the canvas |
| Prism | Free | Low poly triangulated mesh |
| Weave | Free | Geometric tiling patterns (dots, hex, cross) |
| Grain | Pro | Noisy gradient textures |
| Contour | Pro | Topographic elevation line maps |
| Halo | Pro | Radial glow ring effects |
| Cascade | Pro | Stacked angular terraces |

## Stack

- Next.js 14 (App Router)
- Clerk (Auth)
- Convex (Database)
- Tailwind CSS
- Vanilla SVG generation (no dependencies)
- Vercel (Deployment)
